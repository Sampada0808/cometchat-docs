## Using CometChat MCP server

### Claude

To use CometChat MCP server with Claude:

1. **Add MCP server to Claude**
   1. Navigate to the [Connectors](https://claude.ai) page in the Claude settings.
   2. Select **Add custom connector**.
   3. Add CometChat MCP server name and URL.
   4. Select **Add**.

2. **Access CometChat MCP server in your chat**
   1. When using Claude, select the attachments button (the plus icon).
   2. Select CometChat MCP server.
   3. Query Claude with your MCP server as context.

See the [Model Context Protocol documentation](https://modelcontextprotocol.io/) for more details.

### Cursor

To connect your MCP server to Cursor:

1. **Open MCP settings**
   1. Use `Command` + `Shift` + `P` (`Ctrl` + `Shift` + `P` on Windows) to open the command palette.
   2. Search for “Open MCP settings”.
   3. Select **Add custom MCP**. This will open the `mcp.json` file.

2. **Configure CometChat server**

In `mcp.json`, configure CometChat server:

```json
{
  "mcpServers": {
    "server-name": {
      "url": "https://www.cometchat.com/docs/mcp"
    }
  }
}
```

3. **Test the connection**

In Cursor’s chat, ask “How to integrate CometChat in My React App” Cursor should list your documentation search and any configured API endpoints.
