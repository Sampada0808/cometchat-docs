You can integrate the **CometChat Chat Builder widget** directly into your Shopify store to enable real-time chat functionality.

## **What You‚Äôll Build**

You‚Äôll embed a real-time chat widget into your Shopify store using the CometChat Chat Builder. Once set up, it will:
- Initialize the chat widget when the page loads
- Automatically log in a predefined user
- Launch a docked chat interface on your storefront

---

## **Register on CometChat and Get Your Application Keys**

Before integrating CometChat into your Shopify store, you need to create an account and obtain your App ID, Region, and Auth Key.

### **Steps**

1. Go to [CometChat Dashboard](https://app.cometchat.com/) and sign up for a free account.
2. Create a new app by clicking **+ Add App**.
3. Note down the following credentials from the created app:
   - **App ID**
   - **Region**
   - **Auth Key**

These values are used in the integration code provided below.

---

## **Option 1: Embed Widget on All Pages via Shopify Theme**

This method ensures the widget loads on every page of your site.

### **Steps**

1. **Login to Shopify Admin**

   Go to [Shopify Admin](https://admin.shopify.com/) and select your store.

2. **Navigate to Online Store ‚Üí Themes**

   Under **Current theme**, click `Actions ‚Üí Edit Code`.

3. **Open the `theme.liquid` File**

   In the **Layout** folder, click on `theme.liquid`.

4. **Paste Widget Code Before `</body>` Tag**

   Scroll to the bottom and insert the code just before the closing `</body>` tag:

   ```html highlight={14-18, 28}
   <script>
     (function () {
       const script = document.createElement("script");
       script.src =
         "https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@latest/dist/main.js?v=" +
         Date.now();
       script.async = true;
       document.head.appendChild(script);
     })();
   </script>

   <div id="cometChatMount"></div>
   <script>
     const COMETCHAT_CREDENTIALS = {
       appID: "<YOUR_APP_ID>",
       appRegion: "<YOUR_APP_REGION>",
       authKey: "<YOUR_AUTH_KEY>",
     };
     const COMETCHAT_LAUNCH_OPTIONS = {
       targetElementID: "cometChatMount",
       isDocked: true,
       width: "700px",
       height: "400px",
     };

     window.addEventListener("DOMContentLoaded", () => {
       CometChatApp.init(COMETCHAT_CREDENTIALS)
         .then(() => CometChatApp.login({ uid: "<USER_UID>" }))
         .then((user) => CometChatApp.launch(COMETCHAT_LAUNCH_OPTIONS))
         .catch((error) => console.error("[CometChat] Error:", error));
     });
   </script>
   ```

> üìù Replace the placeholder values (`<YOUR_APP_ID>`, `<YOUR_APP_REGION>`, `<YOUR_AUTH_KEY>`, and `<USER_UID>`) with your actual CometChat app credentials.

> ‚ö†Ô∏è Make sure the `uid` (user ID) exists in your CometChat app‚Äôs **Users** tab. You can create test users from the Dashboard.

5. **Click Save**

## **Option 2: Embed Widget on a Specific Page**

If you prefer to show the widget only on one page (like your Contact page):

1. Go to **Online Store ‚Üí Pages** and choose the page.
2. Click the `<>` icon to enter HTML editing mode.
3. Paste the full widget code.
4. Save the page.

> ‚ö†Ô∏è Shopify may restrict custom scripts in content blocks. Test the widget functionality after saving.

## **Option 3: Using Shopify Online Store 2.0 App Blocks (Advanced)**

If your theme supports **Shopify 2.0 app blocks**, you can create a **custom Liquid section** or **app embed block** to manage the widget more modularly.

### **Steps**

1. Go to **Online Store ‚Üí Themes ‚Üí Edit Code**
2. In the **Sections** directory, click **Add a new section** and name it:

```
cometchat-widget.liquid
```

3. Paste the following code:

```liquid highlight={15-19, 29}
{% comment %} CometChat Widget Section {% endcomment %}
<script>
     (function () {
       const script = document.createElement("script");
       script.src =
         "https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@latest/dist/main.js?v=" +
         Date.now();
       script.async = true;
       document.head.appendChild(script);
     })();
</script>

<div id="cometChatMount"></div>
<script>
  const COMETCHAT_CREDENTIALS = {
    appID: "<YOUR_APP_ID>",
    appRegion: "<YOUR_APP_REGION>",
    authKey: "<YOUR_AUTH_KEY>",
  };
  const COMETCHAT_LAUNCH_OPTIONS = {
    targetElementID: "cometChatMount",
    isDocked: true,
    width: "700px",
    height: "400px",
  };

  window.addEventListener("DOMContentLoaded", () => {
    CometChatApp.init(COMETCHAT_CREDENTIALS)
      .then(() => CometChatApp.login({ uid: "<USER_UID>" }))
      .then(() => CometChatApp.launch(COMETCHAT_LAUNCH_OPTIONS))
      .catch((error) => console.error("[CometChat] Error:", error));
  });
</script>
```

4. Save the section and add it to a page template using:

```liquid
{% section 'cometchat-widget' %}
```

> For advanced app-based implementation, refer to [Shopify App Blocks](https://shopify.dev/docs/themes/architecture/sections/app-blocks).

## Troubleshooting

- **Widget not loading?** Ensure your `appID`, `region`, and `authKey` are correct.
- **User login fails?** Confirm the `uid` exists in your CometChat app under Users.
- **Script blocked?** Use the theme integration method instead of inline page content.
