---
title: "Add CometChat Widget to Wix"
description: "Learn how to embed the CometChat Chat Builder widget into your Wix website using custom code embedding."
---

You can integrate the **CometChat Chat Builder widget** into your Wix website using Wix's **Custom Code** feature. Once installed, it will:

- Load and initialize the widget on page load  
- Automatically log in a predefined user  
- Display a docked chat window across your site  

<Steps>
  <Step title="Register on CometChat & Gather Your Keys">
Before you begin, sign up at the [CometChat Dashboard](https://app.cometchat.com/) and create a new app. Copy:
- **App ID**  
- **Region**  
- **Auth Key**
</Step>

<Step title="Add Custom Code in Wix">
1. In your Wix Dashboard, go to **Settings â†’ Custom Code**  
2. Click **+ Add Custom Code**  
3. Paste the following in the **Body - end** field:
```html
<script>
  (function () {
    const script = document.createElement("script");
    script.src =
      "https://cdn.jsdelivr.net/npm/@cometchat/chat-embed@latest/dist/main.js?v=" +
      Date.now();
    script.async = true;
    document.head.appendChild(script);
  })();
</script>

<div id="cometChatMount"></div>
<script>
  const COMETCHAT_CREDENTIALS = {
    appID: "<YOUR_APP_ID>",
    appRegion: "<YOUR_APP_REGION>",
    authKey: "<YOUR_AUTH_KEY>",
  };
  const COMETCHAT_LAUNCH_OPTIONS = {
    targetElementID: "cometChatMount",
    isDocked: true,
    width: "700px",
    height: "400px",
  };

  window.addEventListener("DOMContentLoaded", () => {
    CometChatApp.init(COMETCHAT_CREDENTIALS)
      .then(() => CometChatApp.login({ uid: "<USER_UID>" }))
      .then(() => CometChatApp.launch(COMETCHAT_LAUNCH_OPTIONS))
      .catch((error) => console.error("[CometChat] Error:", error));
  });
</script>
```
4. Set to **All Pages** and choose **Body - end**  
5. Click **Apply**
</Step>

<Step title="Troubleshooting">
- **Widget not visible?** Ensure code is in **Body - end** and enabled on all pages.  
- **Login issues?** Verify the `uid` exists in your CometChat Dashboard under Users.  
- **Still not working?** Reload your site and clear the cache.
</Step>
</Steps>
