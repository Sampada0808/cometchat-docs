You can integrate the **CometChat Visual Builder widget** into your Wix website using Wix's **Custom Code Embed** functionality.

## **What You‚Äôll Build**

You‚Äôll embed a real-time chat widget into your Wix site using the CometChat Visual Builder. Once set up, it will:
- Initialize the chat widget when the page loads
- Automatically log in a predefined user
- Launch a docked chat interface across your site

---

## **Register on CometChat and Get Your Application Keys**

Before integrating CometChat into your Wix site, you need to create an account and obtain your App ID, Region, and Auth Key.

### **Steps**

1. Go to [CometChat Dashboard](https://app.cometchat.com/) and sign up for a free account.
2. Create a new app by clicking **+ Add App**.
3. Note down the following credentials from the created app:
   - **App ID**
   - **Region**
   - **Auth Key**

---

## **Add CometChat Widget Using Custom Code in Wix**

### **Steps**

1. Go to your Wix Dashboard
2. Navigate to **Settings ‚Üí Custom Code**
3. Click **+ Add Custom Code**
4. Paste the following code in the input field:

```html highlight={1, 5-9, 19}
<script src="https://cdn.jsdelivr.net/npm/cometchat-visual-builder-no-code@latest/dist/main.js" defer></script>

<div id="cometChatMount"></div>
<script>
  const COMETCHAT_CREDENTIALS = {
    appID: "<YOUR_APP_ID>",
    appRegion: "<YOUR_APP_REGION>",
    authKey: "<YOUR_AUTH_KEY>",
  };
  const COMETCHAT_LAUNCH_OPTIONS = {
    targetElementID: "cometChatMount",
    isDocked: true,
    width: "700px",
    height: "400px",
  };

  window.addEventListener("DOMContentLoaded", () => {
    CometChatApp.init(COMETCHAT_CREDENTIALS)
      .then(() => CometChatApp.login({ uid: "<USER_UID>" }))
      .then(() => CometChatApp.launch(COMETCHAT_LAUNCH_OPTIONS))
      .catch((error) => console.error("[CometChat] Error:", error));
  });
</script>
```

5. Set the code to appear on **All Pages**, in the **Body - end** section
6. Click **Apply**

> üìù Replace `<YOUR_APP_ID>`, `<YOUR_APP_REGION>`, `<YOUR_AUTH_KEY>`, and `<USER_UID>` with the correct values from your CometChat dashboard.

## Troubleshooting

- **Widget not visible?** Make sure the code is embedded in the **Body - end** and allowed on all pages.
- **Login issues?** Ensure the `uid` is created under the **Users** section of your CometChat app.
- **Still not working?** Try reloading the site and clearing browser cache.
