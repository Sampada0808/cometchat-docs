title: "Notifications"
---

CometChat Notifications come in two variants:

1. The mordern [Notifications](/docs/notifications/overview) platform.

   * [Push Notifications](/docs/notifications/push-overview)
   * [Email Notifications](/docs/notifications/email-overview)
   * [SMS Notification](/docs/notifications/sms-overview)

2. The Legacy Notification Extensions based on CometChat Extensions.

   * [Legacy Push Notifications Extension](/docs/notifications/web-push-notifications)
   * [Legacy Email Notifications Extension](/docs/notifications/email-notification-extension)
   * [Legacy SMS Notifications Extension](/docs/notifications/sms-notification-extension)

For the best experience, we recommend to use the mordern [Notifications](/docs/notifications/overview) platform. Please visit the above mentioned links for more details.
